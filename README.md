# Resume Skill Analysis using Python

This project analyzes 2400 resumes using NLP techniques in Google Colab 
to identify the most frequent skills and aptitudes per category.

## Files
- `DataanalysisveridiabyMK.ipynb` – Colab notebook with code  
- `cleaned_resume.csv` – dataset  
- `Resume_DATA_Analysis_Report_BY_mk.docx` – BRIEF report  
- `images` – For the  visualization  

## Tools
Google Colab, Pandas, Regex, NLTK, spaCy, Matplotlib, Seaborn, Scikit-learn
